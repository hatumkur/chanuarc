# Chanu Arcade — chanuarcade.site

Simple static site for Vercel.

## Deploy to Vercel

1. **Install Vercel CLI** (optional): `npm i -g vercel`
2. **From this folder:** `cd chanuarc` then `vercel`
3. Or **import in Vercel**: [vercel.com/new](https://vercel.com/new) → Import this directory or Git repo.

## Add your domain

1. In Vercel: Project → **Settings** → **Domains**
2. Add **chanuarcade.site** and **www.chanuarcade.site**
3. In Namecheap: **Domain List** → **Manage** → **Advanced DNS**
   - Add the **A** and **CNAME** records Vercel shows (or switch to Vercel nameservers).

After DNS propagates, the site will be live at https://chanuarcade.site
